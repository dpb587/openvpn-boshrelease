openvpn/3.2.0

Released June 8, 2017

Release Notes

  • support pushing DNS servers via openvpn.push_dns
  • support pushing DNS search domains via openvpn.push_dns_search_domains

Usage

Upload the release tarball from the command line:

bosh upload-release --name="openvpn" --version="5.7.0" \
  "https://s3-external-1.amazonaws.com/dpb587-bosh-release-openvpn-us-east-1/artifacts/release/2a077654f266760fd4ca844908664fedeca42ae8" \
  --sha1="sha256:ed5c02e47cb614129a27d741cf67691451c302647668bea4cfc62148ea74b6c8"

Or reference the release from a manifest:

releases:
- name: "openvpn"
  version: "5.7.0"
  url: "https://s3-external-1.amazonaws.com/dpb587-bosh-release-openvpn-us-east-1/artifacts/release/2a077654f266760fd4ca844908664fedeca42ae8"
  sha1: "sha256:ed5c02e47cb614129a27d741cf67691451c302647668bea4cfc62148ea74b6c8"
